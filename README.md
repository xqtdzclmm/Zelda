## 说明


    很多脚本作者写死了助力逻辑，要么只能自助，要么被偷助力，
    本仓库不定期收集各种脚本优化助力逻辑以便更好支持xxoo助力池

    修改了的脚本支持xxoo助力池的 自助，定向助力，随机助力
    
    目前只修改了部分长期活动，如发现有未修改的活动，联系我!!

    


# 特性
- 对接xxoo类助力池
- 需要自助和定向助力的，请参考xxoo助力池说明进行配置,否则只会进行随机助力
- xxoo助力池支持自建服务，所以，本仓库当然也支持切换助力池地址
